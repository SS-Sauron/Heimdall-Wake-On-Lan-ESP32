H̷e̷i̷m̷d̷a̷l̷l̷ Web Flasher

Always watching. Never sleeping. Ready to wake the dead.

Install Heimdall directly to your ESP32 from your browser. Choose one of three presets below.
Check the "Erase device" box during installation if you are flashing for the first time or want to reset your Captive Portal credentials.

shield

Standard

Plug-and-play with all optional features enabled.

  • ▹ Standard MQTT topics
  • ▹ Ping feedback + GPIO + status LED
  • ▹ No TOTP on wake commands
local_police

Hardened (Full)

Full OPSEC plus all optional convenience features.

  • ▹ HMAC topics + TOTP auth
  • ▹ MAC spoof + hostname obfuscation
  • ▹ Ping, GPIO, LED, MQTT responses
visibility_off

Hardened Stealth

OPSEC core with minimal LAN, MQTT, and physical footprint.

  • ▹ Same HMAC + TOTP + identity as Hardened
  • ▹ No status LED, ping, or GPIO
  • ▹ No MQTT wake confirmations or presence

Stealth reduces observability — you will not receive wake confirmation over MQTT.

warning Troubleshooting

Device not showing up in the list?

  • usb
    Cables: You are likely using a charge-only USB cable. You MUST use a data cable. Try another cable first!
  • memory
    Drivers: Your computer might be missing the USB-to-UART drivers. Download them here:
    Silabs CP2102 WCH CH340
  • public
    Browser: This feature uses the Web Serial API. It only works in Chromium-based browsers like Google Chrome, Microsoft Edge, or Brave.
